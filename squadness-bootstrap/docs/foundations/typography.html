Foundations

Tipografía

Familias, escala de tamaños y pesos del sistema. Todos los tokens están en tokens.css como propiedades --sq-* y sobreescriben las variables de Bootstrap automáticamente.

Familias de fuente

Heading
--sq-font-heading
Pangram — Aa Bb Cc 0123
Fuente exclusiva para títulos y headings (h1–h6). Fuente comercial — debe incluirse mediante @font-face con los archivos de licencia.
Body
--sq-font-body
Inter — Aa Bb Cc 0123
Fuente principal para textos de interfaz, labels, botones e inputs. Cargada desde Google Fonts con pesos 400, 500, 600, 700.
Monospace
--sq-font-mono
Roboto Mono 0123
Bloques de código, tokens CSS, valores técnicos. Cargada desde Google Fonts con pesos 400, 500.

Carga Inter y Roboto Mono desde Google Fonts. Pangram es una fuente comercial — añádela con @font-face apuntando a tus archivos locales de licencia.

<!-- Google Fonts -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Roboto+Mono:wght@400;500&display=swap" rel="stylesheet" />

<!-- Pangram (fuente comercial — archivo local) -->
<style>
  @font-face {
    font-family: 'Pangram';
    src: url('/fonts/Pangram-Regular.woff2') format('woff2');
    font-weight: 400; font-style: normal;
  }
  /* Repite por cada peso que uses */
</style>
