Espaciado

Escala de espaciado, radios de borde y sombras del sistema. Usa los tokens --sq-space-* en CSS o las clases de utilidad Bootstrap (.p-*, .m-*, .gap-*) mapeadas a la escala de 8 puntos.

Escala de espaciado

Sistema basado en 8px. Cada token es un múltiplo o submúltiplo del step base de 8px.

--sq-space-0 0px Reset de margin/padding, alineación flush
--sq-space-025 2px Separación mínima entre badge y texto adyacente
--sq-space-050 4px Gap entre icono y label, separadores inline
--sq-space-075 6px Padding horizontal de badges pequeños
--sq-space-100 8px Step base. Padding de botones sm, gap de form check
--sq-space-150 12px Padding vertical de inputs, gap de nav items
--sq-space-200 16px Padding de card, separación entre elementos de form
--sq-space-250 20px Gap de columnas de grid, margen de sección
--sq-space-300 24px Padding de card body, separación entre títulos
--sq-space-400 32px Margen entre secciones, separación de grupos
--sq-space-500 40px Padding de secciones de página, hero padding
--sq-space-600 48px Separación grande entre bloques de contenido
--sq-space-800 64px Separación de alto nivel, section breaks

Mapeo con Bootstrap

Bootstrap usa su propia escala de espaciado $spacer (base 1rem = 16px) para las clases .p-*, .m-*, .gap-*. Referencia de equivalencia:

Clase BootstrapValorToken equivalenteUso sugerido
.p-1 / .m-14px (0.25rem)--sq-space-050Gap de iconos, padding xs
.p-2 / .m-28px (0.5rem)--sq-space-100Padding de botones sm
.p-3 / .m-316px (1rem)--sq-space-200Padding de card, separación de campos
.p-4 / .m-424px (1.5rem)--sq-space-300Padding de card body
.p-5 / .m-548px (3rem)--sq-space-600Separación de bloques grandes

Radios de borde

--sq-radius-xs 4px Badges, checkboxes, tooltips, focus rings. --bs-border-radius-sm
--sq-radius-sm 8px Botones, inputs, nav items. --bs-border-radius
--sq-radius-md 12px Dropdowns, popovers. --bs-border-radius-lg
--sq-radius-lg 16px Cards, modales, offcanvas. --bs-border-radius-xl
--sq-radius-full 9999px Pill badges, avatar, switches, tags. --bs-border-radius-pill
