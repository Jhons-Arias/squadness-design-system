Componentes

Badge

Etiqueta visual para comunicar estado, categoría o cantidad. Estilo outlined pill: borde coloreado + fondo sutil. Siete variantes semánticas.

Variantes

Usa .badge.bg-{variant}. Cada variante tiene borde, fondo y texto propios definidos en squadness.css.

Con contador

Badge posicionado como notificación sobre un elemento.

<button class="btn btn-secondary position-relative">
  Notificaciones
  <span class="position-absolute top-0 start-100 translate-middle badge bg-danger"
        style="font-size:10px;">
    3
  </span>
</button>

Badge integrado dentro de encabezados y párrafos.
