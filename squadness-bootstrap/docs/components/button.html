Componentes

Button

Usa los botones de Squadness para acciones en formularios, diálogos y más. Cinco variantes, tres tamaños y soporte completo de estados.

Variantes

Squadness incluye 5 variantes, cada una con un propósito semántico definido.

<button type="button" class="btn btn-primary">Primary</button>
<button type="button" class="btn btn-secondary">Secondary</button>
<button type="button" class="btn btn-outline-secondary">Outline</button>
<button type="button" class="btn btn-ghost">Ghost</button>
<button type="button" class="btn btn-danger">Delete</button>

Tamaños

Usa .btn-lg o .btn-sm para tamaños adicionales.

<button type="button" class="btn btn-primary btn-lg">Large button</button>
<button type="button" class="btn btn-secondary btn-lg">Large button</button>
<button type="button" class="btn btn-outline-secondary btn-lg">Large button</button>
<button type="button" class="btn btn-primary">Default button</button>
<button type="button" class="btn btn-secondary">Default button</button>
<button type="button" class="btn btn-outline-secondary">Default button</button>
<button type="button" class="btn btn-primary btn-sm">Small button</button>
<button type="button" class="btn btn-secondary btn-sm">Small button</button>
<button type="button" class="btn btn-outline-secondary btn-sm">Small button</button>

Estado deshabilitado

Haz que los botones luzcan inactivos agregando el atributo disabled. Para <a>, usa la clase .disabled junto con aria-disabled="true".

<!-- Botones HTML — atributo disabled -->
<button type="button" class="btn btn-primary" disabled>Primary</button>
<button type="button" class="btn btn-secondary" disabled>Secondary</button>
<button type="button" class="btn btn-outline-secondary" disabled>Outline</button>
<button type="button" class="btn btn-ghost" disabled>Ghost</button>
<button type="button" class="btn btn-danger" disabled>Delete</button>

<!-- Links — clase .disabled -->
<a class="btn btn-primary disabled" aria-disabled="true">Primary</a>

Estado activo

Fuerza el estado de presionado con la clase .active.

<button type="button" class="btn btn-primary active">Primary</button>
<button type="button" class="btn btn-secondary active">Secondary</button>
<button type="button" class="btn btn-outline-secondary active">Outline</button>
<button type="button" class="btn btn-ghost active">Ghost</button>
<button type="button" class="btn btn-danger active">Delete</button>

Estado loading

Combina el spinner de Bootstrap con disabled para indicar que una acción está en progreso.

<button type="button"
        class="btn btn-primary d-inline-flex align-items-center gap-2"
        disabled>
  <span class="spinner-border"
         style="width:14px;height:14px;border-width:2px;"
         role="status" aria-hidden="true"></span>
  Guardando...
</button>

Con iconos

Agrega d-inline-flex align-items-center gap-1 para alinear un ícono SVG con el texto del botón.

<!-- Ícono izquierda -->
<button type="button" class="btn btn-primary d-inline-flex align-items-center gap-1">
  <svg width="14" height="14">...</svg>
  Nuevo
</button>

<!-- Ícono derecha -->
<button type="button" class="btn btn-secondary d-inline-flex align-items-center gap-1">
  Exportar
  <svg width="14" height="14">...</svg>
</button>

Ancho completo

Usa la utilidad .w-100 de Bootstrap para que el botón ocupe todo el ancho de su contenedor.

<button type="button" class="btn btn-primary w-100">Guardar cambios</button>
<button type="button" class="btn btn-outline-secondary w-100">Cancelar</button>
