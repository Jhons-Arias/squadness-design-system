Navbar

Barra de navegación superior (Top Bar). Contiene el toggle de sidebar, notificaciones y la información del usuario activo. Clase principal: .sq-topbar.

Top Bar

Barra de 56px a ancho completo con border-bottom. Izquierda: botón ghost para colapsar el sidebar (.sq-topbar-icon-btn). Derecha: selector de tema, botón de notificaciones (.sq-topbar-icon-btn--outline), divisor vertical y datos del usuario activo. Clic en el usuario para ver el estado Active.
