Toast

Notificación compacta no intrusiva. Usa la clase .sq-toast junto con el modificador de variante (.sq-toast--success, --warning, --info, --error). Para el comportamiento de show/hide usa Bootstrap .toast en el mismo elemento.

Variantes

Cinco variantes semánticas. Las variantes con ícono usan gap: 8px y el SVG hereda el color del texto mediante currentColor. La variante Default no lleva ícono y tiene gap: 0.

Demo interactivo

Combina .sq-toast con la clase .toast de Bootstrap en el mismo elemento para obtener el comportamiento de show/hide animado. El toast aparece en la esquina inferior derecha.

<!-- Contenedor de toasts — esquina inferior derecha -->
<div class="toast-container position-fixed bottom-0 end-0 p-3" style="z-index:1090">

  <div id="myToast"
       class="toast sq-toast sq-toast--success"
       role="alert" aria-live="assertive" aria-atomic="true"
       data-bs-delay="4000">
    <svg class="sq-toast-icon" width="16" height="16" viewBox="0 0 16 16" fill="none">
      <circle cx="8" cy="8" r="6.5" stroke="currentColor" stroke-width="1.4"/>
      <path d="M5 8l2.5 2.5L11 5.5" stroke="currentColor" stroke-width="1.4" stroke-linecap="round"/>
    </svg>
    <span class="sq-toast-text">Cambios guardados correctamente</span>
  </div>

</div>

<!-- Trigger -->
<button onclick="new bootstrap.Toast(document.getElementById('myToast')).show()">
  Mostrar toast
</button>
