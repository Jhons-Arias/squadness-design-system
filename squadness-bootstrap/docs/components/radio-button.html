Radio Button

Control de selección única dentro de un grupo. Todos los radios de un grupo deben compartir el mismo atributo name.

Grupo básico

Clases clave: .form-check .form-check-input[type="radio"]. Todos los radios de un grupo deben compartir el mismo name="".

Radio Button Date

Variante especializada para filtros de período temporal. Ofrece opciones rápidas predefinidas y una opción "Fechas personalizadas" que expande dos campos de fecha. Clase base: .sq-radio-date.
