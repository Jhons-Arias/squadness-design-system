Componentes

Switch

Control de activación on/off. Reemplaza al checkbox cuando la acción tiene efecto inmediato sin necesidad de confirmar.

Básico

Clases clave: .form-check.form-switch .form-check-input[type="checkbox"][role="switch"].

<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="switch-basic" />
  <label class="form-check-label" for="switch-basic">Activar notificaciones</label>
</div>

<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="switch-on" checked />
  <label class="form-check-label" for="switch-on">Modo oscuro</label>
</div>

Deshabilitado

Agrega el atributo disabled para bloquear la interacción.

<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="switch-off-disabled" disabled />
  <label class="form-check-label" for="switch-off-disabled" style="color:var(--sq-text-subtlest);">
    Sincronización en tiempo real (no disponible)
  </label>
</div>
