Select

Lista desplegable para selección de opciones predefinidas. Soporta selección única, múltiple y estados de validación.

Básico

  <button class="sq-select-toggle" type="button"
          data-bs-toggle="dropdown" aria-expanded="false">
    <span class="sq-select-label sq-select-placeholder">Seleccionar...</span>
    <svg class="sq-select-caret" viewBox="0 0 16 16" fill="none">
      <path d="M4 6l4 4 4-4" stroke="currentColor" stroke-width="1.5" stroke-linecap="round"/>
    </svg>
  </button>
  <ul class="dropdown-menu w-100">
    <li><button class="dropdown-item" type="button" data-value="active">Activo</button></li>

    <li><button class="dropdown-item" type="button" data-value="pending">Pendiente</button></li>

  </ul>
</div>

Menú desplegable

El menú contextual usa .dropdown-menu de Bootstrap con overrides Squadness. El ítem seleccionado lleva .active con fondo brand sutil (#cce2ff) y texto oscuro, más un checkmark de 16px al final.
