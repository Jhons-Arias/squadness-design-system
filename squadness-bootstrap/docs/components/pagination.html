Componentes

Pagination

Componente de navegación entre páginas. Combina el selector de filas por página con el contador de página actual y los botones de navegación (.sq-icon-btn). Usa la clase .sq-pagination como contenedor.

Pagination

Dos grupos alineados en fila: el selector de rows per page a la izquierda y el contador de página con los botones de navegación a la derecha. Gap de 16px entre grupos, 8px entre elementos dentro de cada grupo.

Estados de navegación

Los botones de navegación usan el componente .sq-icon-btn. Cuando se está en la primera página, el botón anterior se deshabilita con el atributo disabled; en la última, el siguiente.
