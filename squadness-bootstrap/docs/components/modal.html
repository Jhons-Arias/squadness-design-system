Componentes

Modal

Diálogos de confirmación, formulario y visualización de imágenes. Usa .sq-modal como contenedor visual y Bootstrap .modal para la capa de overlay y accesibilidad.

Eliminar filas

Modal de confirmación destructiva. Muestra título, descripción y dos acciones: Cancelar con .sq-btn-cancel y Eliminar filas con .sq-btn-danger (fondo #9e0b15, texto #feeded).

<!-- Trigger -->
<button data-bs-toggle="modal" data-bs-target="#modalEliminar">
  Eliminar filas
</button>

<!-- Modal -->
<div class="modal fade" id="modalEliminar" tabindex="-1">
  <div class="modal-dialog modal-dialog-centered sq-modal-dialog">
    <div class="modal-content">
      <div class="sq-modal">

        <div class="sq-modal-header">
          <h2 class="sq-modal-title">Eliminar filas</h2>
          <button class="sq-icon-btn sq-icon-btn-outline"
                  data-bs-dismiss="modal" aria-label="Cerrar">
            <!-- close SVG 16px -->
          </button>
        </div>

        <p class="sq-modal-body">
          ¿Estás seguro que deseas eliminar las filas seleccionadas?
          Esta acción no se puede deshacer.
        </p>

          <button class="sq-btn-cancel" data-bs-dismiss="modal">Cancelar</button>

  </div>
</div>

Adicionar a los baños

Modal con formulario. Agrega .sq-modal--form para aumentar el gap entre bloques a 32 px. Los campos usan .sq-modal-input (h 40 px, radius 6 px) y el selector usa .sq-modal-dropdown (h 36 px, radius 12 px). La acción principal usa .sq-btn-primary-action.

<button data-bs-toggle="modal" data-bs-target="#modalAdicionar">

Lightbox de imagen

Modal de visualización ampliada con carrusel. El trigger es un .sq-images-container con thumbnails .sq-image-thumb (124×167 px). Al hacer click en cualquier imagen se abre el lightbox posicionado en ese índice. Usa Bootstrap .modal con data-bs-backdrop="false" para que el propio .modal-content actúe como overlay oscuro.

<!-- Grid de thumbnails — cada uno abre el lightbox en su índice -->
<div class="sq-images-container">
  <div class="sq-image-thumb">
    <img src="thumb1.jpg" data-full-src="full1.jpg" alt="Foto 1" />
  </div>
  <div class="sq-image-thumb">
    <img src="thumb2.jpg" data-full-src="full2.jpg" alt="Foto 2" />
  </div>
  <!-- más thumbnails... -->
</div>

<!-- Modal lightbox (una instancia global en el documento) -->
<div class="modal fade sq-lightbox" id="sqLightbox"
     tabindex="-1" data-bs-backdrop="false">
  <div class="modal-dialog modal-fullscreen">
    <div class="modal-content">
      <button class="sq-icon-btn sq-icon-btn-outline sq-lightbox-close"
              data-bs-dismiss="modal" aria-label="Cerrar">
        <!-- close SVG 16px -->
      </button>
      <div class="sq-lightbox-carousel">
        <button class="sq-lightbox-nav" id="sqLbPrev">
          <!-- chevron-left SVG 24px stroke white -->
        </button>
        <div class="sq-lightbox-frame">
          <img id="sqLbImg" class="sq-lightbox-img" src="" alt="" />
        </div>
        <button class="sq-lightbox-nav" id="sqLbNext">
          <!-- chevron-right SVG 24px stroke white -->
        </button>
      </div>
      <span class="sq-lightbox-counter" id="sqLbCounter"></span>
    </div>
  </div>
</div>

Delete Dialog

Barra de acción contextual que aparece al seleccionar filas en una tabla. Muestra el conteo de selección, una acción secundaria outline para seleccionar todo y un botón destructivo con icono de papelera. Clase principal: .sq-delete-dialog.

Se seleccionaron 1 fila 
<div class="sq-delete-dialog">

  <div class="sq-delete-dialog-info">
    <span class="sq-delete-dialog-count">Se seleccionaron 1 fila</span>
    <button class="sq-delete-dialog-select-all">
      Seleccionar las 32 baños
    </button>
  </div>

  <button class="sq-delete-dialog-btn-danger">
    Delete
    <svg width="16" height="16" viewBox="0 0 24 24" fill="currentColor">
      <path d="M7 21q-.825 0-1.412-.587T5 19V6H4V4h5V3h6v1h5v2h-1v13q0 .825-.587 1.413T17 21H7Zm2-4h2V8H9v9Zm4 0h2V8h-2v9Z"/>
    </svg>
  </button>

</div>
