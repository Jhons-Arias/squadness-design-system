Componentes

Sidebar

Panel de navegación lateral. Contiene logo, lista de ítems y grupos de submenú. Soporta dos variantes: expandido (240 px) y colapsado (56 px, solo íconos). Clase principal: .sq-sidebar.

Sidebar

Sidebar expandido con su contenido real de ejemplo. Usa el botón de la cabecera para alternar la variante colapsada. El grupo Baños Portátiles está expandido por defecto mostrando el subítem activo.

Subcomponente de ítem con submenú. .sq-sidebar-group sin modificador = colapsado. Agrega .is-open para expandirlo. El chevron rota 180° y el submenú aparece con una línea vertical de acento.

Colapsado

La clase modificadora .sq-sidebar--collapsed estrecha el sidebar a 56 px y oculta las etiquetas de texto. Los íconos se mantienen visibles y centrados. Usa title en los ítems para tooltips nativos al hacer hover.

API de clases

ClaseDescripción
.sq-sidebarContenedor principal — 240 px, borde derecho, fondo surface-default
.sq-sidebar--collapsedVariante colapsada — 56 px, etiquetas ocultas, íconos centrados visibles
.sq-sidebar-headerCabecera 56 px con borde inferior; contiene logo y wordmark
.sq-sidebar-logoÍcono del logo (32 px); siempre visible
.sq-sidebar-wordmarkNombre de la marca; se oculta con --collapsed
.sq-sidebar-toggleBotón de colapso/expansión (24 px, borde, ghost)
.sq-sidebar-menuLista de ítems — flex-col, gap 4 px, padding 24 px 12 px
.sq-sidebar-itemÍtem simple (ícono + etiqueta); modificador .active
.sq-sidebar-iconSVG inline del ícono (16×16 px); hereda el color del padre con fill="currentColor"
.sq-sidebar-item-labelTexto del ítem; se oculta con --collapsed
.sq-sidebar-groupGrupo acordeón; .is-open para expandir
.sq-sidebar-group-triggerCabecera clickeable del grupo (ícono + label + chevron)
.sq-sidebar-group-labelTexto del grupo; se oculta con --collapsed
.sq-sidebar-group-chevronSVG chevron — rota 180° cuando el grupo tiene .is-open
.sq-sidebar-group-subSubmenú oculto; se muestra con .is-open en el grupo
.sq-sidebar-sub-dividerLínea vertical (1 px) como acento visual del submenú
.sq-sidebar-sub-listLista de subítems
.sq-sidebar-sub-itemSubítem; modificador .active (fondo brand azul, texto blanco)
