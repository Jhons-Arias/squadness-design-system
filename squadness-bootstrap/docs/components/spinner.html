Componentes

Spinner

Indicador de carga para procesos en segundo plano. Puede usarse en botones, como overlay o como estado vacío.

Border spinner

Spinner de anillo giratorio. El tipo más común para estados de carga.

Cargando... 
<div class="spinner-border" role="status">
  <span class="visually-hidden">Cargando...</span>
</div>

Grow spinner

Spinner que pulsa y crece. Alternativa visual más suave al border spinner.

Cargando... 
<div class="spinner-grow" role="status">
  <span class="visually-hidden">Cargando...</span>
</div>

Tamaños

Usa .spinner-border-sm para el tamaño pequeño, o controla el tamaño grande con style.

<!-- Pequeño -->
<div class="spinner-border spinner-border-sm" role="status" aria-hidden="true"></div>

<!-- Default -->
<div class="spinner-border" role="status" aria-hidden="true"></div>

<!-- Grande -->
<div class="spinner-border" style="width:2.5rem;height:2.5rem;" role="status" aria-hidden="true"></div>

Colores semánticos

El spinner hereda el color de texto. Usa variables de token o clases de Bootstrap para variantes de color.

<div class="spinner-border" style="color:var(--sq-brand);" role="status" aria-hidden="true"></div>
<div class="spinner-border text-success" role="status" aria-hidden="true"></div>
<div class="spinner-border text-warning" role="status" aria-hidden="true"></div>
<div class="spinner-border text-danger" role="status" aria-hidden="true"></div>
<div class="spinner-border" style="color:var(--sq-text-subtlest);" role="status" aria-hidden="true"></div>

En botón

Coloca el spinner dentro de un botón deshabilitado para indicar que una acción está en proceso.

<button class="btn btn-primary" disabled>
  <span class="spinner-border spinner-border-sm me-2"
        style="width:14px;height:14px;border-width:2px;"
        role="status" aria-hidden="true"></span>
  Guardando...
</button>

<button class="btn btn-secondary" disabled>
  <span class="spinner-border spinner-border-sm me-2"
        style="width:14px;height:14px;border-width:2px;"
        role="status" aria-hidden="true"></span>
  Exportando...
</button>

Estado de carga de página

Spinner centrado con texto para reemplazar secciones completas mientras se cargan datos.

Cargando...

Cargando información...

Cargando registros... 
<!-- Estado de carga de página completa -->
<div class="d-flex flex-column align-items-center justify-content-center gap-3"
     style="padding:64px 0;">
  <div class="spinner-border" style="width:2rem;height:2rem;" role="status">
    <span class="visually-hidden">Cargando...</span>
  </div>
  <p style="font-size:14px;color:var(--sq-text-subtlest);margin:0;">Cargando información...</p>
</div>

<!-- Estado de carga dentro de una card -->
<div class="card" style="max-width:400px;">
  <div class="card-body d-flex align-items-center justify-content-center gap-3"
       style="min-height:120px;">
    <div class="spinner-border spinner-border-sm" role="status" aria-hidden="true"></div>
    <span style="font-size:14px;color:var(--sq-text-subtlest);">Cargando registros...</span>
  </div>
</div>

Overlay de carga

Spinner superpuesto sobre contenido existente usando position-absolute sobre un contenedor position-relative.

Contenido de la sección...

Cargando...
Procesando... 
<!-- Overlay de carga sobre un contenido -->
<div class="position-relative" style="min-height:160px;">
  <!-- Contenido que se oscurece -->
  <div style="opacity:0.3;pointer-events:none;">
    <p>Contenido de la sección...</p>
    <button class="btn btn-primary btn-sm">Acción</button>
  </div>
  <!-- Overlay con spinner -->
  <div class="position-absolute top-0 start-0 w-100 h-100 d-flex align-items-center justify-content-center"
       style="background:rgba(248,250,252,0.8);border-radius:var(--sq-radius-lg);">
    <div class="d-flex flex-column align-items-center gap-2">
      <div class="spinner-border" role="status">
        <span class="visually-hidden">Cargando...</span>
      </div>
      <span style="font-size:13px;color:var(--sq-text-subtlest);">Procesando...</span>
    </div>
  </div>
</div>
