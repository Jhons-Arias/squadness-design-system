Popover

Panel flotante de filtros avanzados. Incluye selects con búsqueda, calendario de fecha y botón de acción.

Filtro avanzado

Clic en un campo select para abrir el menú de opciones; clic en un ítem para seleccionarlo. Clic en el campo Fecha para abrir el calendario. Navega entre meses con ← →.
