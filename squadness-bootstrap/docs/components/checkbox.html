Componentes

Checkbox

Control de selección múltiple. Soporta estado marcado, desmarcado, deshabilitado e indeterminado.

Básico

Checkbox estándar sin marcar y marcado por defecto. Clases clave: .form-check .form-check-input .form-check-label.

<div class="form-check">
  <input class="form-check-input" type="checkbox" id="check-basic" />
  <label class="form-check-label" for="check-basic">
    Aceptar términos y condiciones
  </label>
</div>

<div class="form-check">
  <input class="form-check-input" type="checkbox" id="check-checked" checked />
  <label class="form-check-label" for="check-checked">
    Recibir notificaciones por email
  </label>
</div>

Deshabilitado

Agrega el atributo disabled al input para bloquear la interacción.

<div class="form-check">
  <input class="form-check-input" type="checkbox" id="check-disabled" disabled />
  <label class="form-check-label" for="check-disabled" style="color:var(--sq-text-subtlest);">
    Opción no disponible
  </label>
</div>

<div class="form-check">
  <input class="form-check-input" type="checkbox" id="check-disabled-checked" checked disabled />
  <label class="form-check-label" for="check-disabled-checked" style="color:var(--sq-text-subtlest);">
    Permiso heredado (no editable)
  </label>
</div>

Grupo de checkboxes

Usa un <fieldset> con <legend> para agrupar opciones relacionadas.

Permisos del rol 
<fieldset>
  <legend class="form-label">Permisos del rol</legend>
  <div class="d-flex flex-column gap-2">
    <div class="form-check">
      <input class="form-check-input" type="checkbox" id="perm-read" checked />
      <label class="form-check-label" for="perm-read">Ver registros</label>
    </div>
    <div class="form-check">
      <input class="form-check-input" type="checkbox" id="perm-create" checked />
      <label class="form-check-label" for="perm-create">Crear registros</label>
    </div>
    <div class="form-check">
      <input class="form-check-input" type="checkbox" id="perm-edit" />
      <label class="form-check-label" for="perm-edit">Editar registros</label>
    </div>
    <div class="form-check">
      <input class="form-check-input" type="checkbox" id="perm-delete" />
      <label class="form-check-label" for="perm-delete">Eliminar registros</label>
    </div>
  </div>
</fieldset>

Inline

    <input class="form-check-input" type="checkbox" id="inline-1" checked />

  </div>

  </div>
