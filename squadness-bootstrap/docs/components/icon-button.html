Icon Button

Botón cuadrado de acción que contiene únicamente un ícono SVG. Seis variantes: Primary, Secondary, Tertiary, Outline, Ghost y Danger.

Variantes

Usa .btn-icon + .btn-icon-{variant}. Tamaño 32×32 px, border-radius 12 px, ícono 16×16 px.

<!-- Primary -->
<button type="button" class="btn-icon btn-icon-primary" aria-label="Acción">
  <svg viewBox="0 0 16 16">...</svg>
</button>

<!-- Secondary -->
<button type="button" class="btn-icon btn-icon-secondary" aria-label="Acción">...</button>

<!-- Tertiary -->
<button type="button" class="btn-icon btn-icon-tertiary" aria-label="Acción">...</button>

<!-- Outline -->
<button type="button" class="btn-icon btn-icon-outline" aria-label="Acción">...</button>

<!-- Ghost -->
<button type="button" class="btn-icon btn-icon-ghost" aria-label="Acción">...</button>

<!-- Danger -->
<button type="button" class="btn-icon btn-icon-danger" aria-label="Eliminar">...</button>

Estado deshabilitado

Agrega el atributo disabled. Opacity 0.45, sin interacción.

<button type="button" class="btn-icon btn-icon-primary" disabled>...</button>

En toolbars

Patrón común: grupo de Icon Buttons para acciones de tabla o editor.

<div class="d-flex align-items-center gap-1">
  <button type="button" class="btn-icon btn-icon-ghost" aria-label="Editar">
    <svg viewBox="0 0 16 16">...</svg>
  </button>
  <button type="button" class="btn-icon btn-icon-danger" aria-label="Eliminar">
    <svg viewBox="0 0 16 16">...</svg>
  </button>
</div>
