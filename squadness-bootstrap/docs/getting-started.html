Getting Started

Cómo instalar

Integra el Design System en tu proyecto PHP, Laravel, Django o cualquier stack que genere HTML en 3 pasos.

Paso 1 — Copiar los archivos CSS

Copia la carpeta src/ a la carpeta de assets de tu proyecto. Por ejemplo, en Laravel sería public/assets/squadness/.

tu-proyecto/
└── public/
    └── assets/
        └── squadness/
            ├── tokens.css
            └── squadness.css

Paso 2 — Incluir en el <head>

El orden de los imports es crítico. Bootstrap debe cargarse después de tokens.css para que las sobreescrituras funcionen.

<!-- Google Fonts (Inter + Roboto Mono) -->
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Roboto+Mono:wght@400;500&display=swap" rel="stylesheet" />

<!-- 1. Squadness Tokens (--sq-* y override --bs-*) -->
<link rel="stylesheet" href="/assets/squadness/tokens.css" />

<!-- 2. Bootstrap 5 CDN -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" />

<!-- 3. Squadness Components skin -->
<link rel="stylesheet" href="/assets/squadness/squadness.css" />
Bootstrap 5 puede cargarse también vía npm (npm install bootstrap). El sistema de tokens funciona igual independientemente del método de carga.

Paso 3 — Usar componentes Bootstrap

Usa las clases Bootstrap de siempre. El Design System se aplica automáticamente via las variables CSS.

<!-- Los botones Bootstrap adoptan la paleta Squadness automáticamente -->
<button class="btn btn-primary">Guardar cambios</button>
<button class="btn btn-secondary">Cancelar</button>

<!-- Las cards usan el radio y sombra del DS -->
<div class="card">
  <div class="card-body">Contenido</div>
</div>

<!-- Las tablas tienen el estilo de thead del DS -->
<table class="table table-hover">...</table>

Template base PHP

Template mínimo para una aplicación PHP con el DS integrado.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title><?= htmlspecialchars($pageTitle ?? 'Mi App') ?></title>

  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet" />

  <link rel="stylesheet" href="/assets/squadness/tokens.css" />
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" />
  <link rel="stylesheet" href="/assets/squadness/squadness.css" />
  <link rel="stylesheet" href="/assets/css/app.css" /> <!-- tu CSS personalizado -->
</head>
<body>

  <?php include 'partials/navbar.php'; ?>

  <div class="container-fluid">
    <div class="row">
      <?php include 'partials/sidebar.php'; ?>

      <main class="col">
        <div class="py-4 px-3">
          <?= $content ?>
        </div>
      </main>
    </div>
  </div>

  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
  <script src="/assets/js/app.js"></script>
</body>
</html>

Integración con Laravel + Blade

{{-- resources/views/layouts/app.blade.php --}}
<!DOCTYPE html>
<html lang="{{ str_replace('_', '-', app()->getLocale()) }}">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>@yield('title', config('app.name'))</title>

  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet" />

  <link rel="stylesheet" href="{{ asset('css/tokens.css') }}" />
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" />
  <link rel="stylesheet" href="{{ asset('css/squadness.css') }}" />
  @stack('styles')
</head>
<body>
  @include('partials.navbar')

  <div class="container py-4">
    @yield('content')
  </div>

  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
  @stack('scripts')
</body>
</html>
