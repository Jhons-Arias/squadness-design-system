Getting Started

Integración PHP

Cómo integrar Squadness Bootstrap en proyectos PHP puros y en aplicaciones Laravel + Blade. Incluye estructura de carpetas, layout base, partials reutilizables, y un ejemplo de CRUD completo.

Estructura de carpetas

Copia la carpeta src/ del repositorio dentro de tus assets públicos. Solo necesitas tres archivos CSS.

tu-proyecto/
├── public/
│   └── assets/
│       ├── squadness/
│       │   ├── tokens.css          ← tokens --sq-* + overrides --bs-*
│       │   └── squadness.css       ← estilos de componentes
│       └── css/
│           └── app.css             ← tus estilos propios
├── partials/
│   ├── head.php                    ← <head> con imports
│   ├── navbar.php                  ← topbar / navbar
│   ├── sidebar.php                 ← sidebar de navegación
│   └── scripts.php                 ← JS al final del body
├── pages/
│   ├── dashboard.php
│   ├── users/
│   │   ├── index.php               ← listado
│   │   ├── create.php              ← formulario nuevo
│   │   └── edit.php                ← formulario edición
└── index.php
Alternativa dist/: Usa dist/squadness-bootstrap.min.css (25 KB) como único archivo CSS, reemplazando tanto tokens.css como squadness.css. No incluye Bootstrap — sigue necesitando bootstrap.min.css.

Laravel + Blade

Ejemplo completo del layout maestro Blade con soporte para stacks de CSS y JS.

<?php
// pages/users/save.php — procesa crear y editar

session_start();
require_once '../../config/db.php';

// CSRF
if (!isset($_POST['_token']) || $_POST['_token'] !== $_SESSION['csrf']) {
  http_response_code(403);
  exit('CSRF mismatch');
}

$id    = (int)($_POST['id'] ?? 0);
$name  = trim($_POST['name'] ?? '');
$email = filter_var($_POST['email'] ?? '', FILTER_VALIDATE_EMAIL);
$role  = in_array($_POST['role'], ['admin','editor','viewer']) ? $_POST['role'] : null;
$active = (int)($_POST['active'] ?? 1);

// Validación básica
$errors = [];
if (!$name)    $errors[] = 'El nombre es requerido.';
if (!$email)   $errors[] = 'Email inválido.';
if (!$role)    $errors[] = 'Rol inválido.';

if ($errors) {
  $_SESSION['flash_error'] = implode(' ', $errors);
  header('Location: index.php');
  exit;
}

if ($id === 0) {
  // Crear
  $password = password_hash($_POST['password'] ?? '', PASSWORD_DEFAULT);
  $stmt = $pdo->prepare(
    'INSERT INTO users (name, email, role, active, password, created_at)
     VALUES (?, ?, ?, ?, ?, NOW())'
  );
  $stmt->execute([$name, $email, $role, $active, $password]);
  $_SESSION['flash_success'] = 'Usuario creado correctamente.';
} else {
  // Editar
  $stmt = $pdo->prepare(
    'UPDATE users SET name=?, email=?, role=?, active=?, updated_at=NOW()
     WHERE id=?'
  );
  $stmt->execute([$name, $email, $role, $active, $id]);
  $_SESSION['flash_success'] = 'Usuario actualizado.';
}

header('Location: index.php');
exit;

Flash messages con Session

Patrón para mostrar alertas de éxito/error después de un redirect POST.

¡Listo! Usuario creado correctamente.

<?php // partials/flash.php
session_start();
?>
<?php if (!empty($_SESSION['flash_success'])): ?>

    <strong>¡Listo!</strong> <?= htmlspecialchars($_SESSION['flash_success']) ?>

  </div>

<?php endif; ?>

Usar dist/ en producción

Para producción puedes usar el bundle pre-compilado de dist/ y reducir el número de requests HTTP.

<!-- Producción: 1 solo archivo Squadness (25 KB minificado) -->
<link rel="stylesheet" href="/assets/dist/squadness-bootstrap.min.css" />
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" />

<!-- Nota: el orden importa.
     squadness-bootstrap.min.css debe ir ANTES de bootstrap.min.css -->
ArchivoTamañoContenido
dist/squadness-bootstrap.css 44 KB Bundle completo sin minificar (desarrollo/debugging)
dist/squadness-bootstrap.min.css 25 KB Bundle minificado para producción (42% más pequeño)
