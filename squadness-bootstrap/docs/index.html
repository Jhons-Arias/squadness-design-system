Getting Started

Introducción

Squadness Design System para Bootstrap 5 es una capa CSS que implementa los tokens de diseño Squadness sobre Bootstrap. Compatible con PHP, Laravel, Django y cualquier stack que genere HTML.

Principios de diseño

Consistencia
Todos los componentes comparten los mismos tokens de color, espaciado y tipografía.
Single source of truth
Los mismos --sq-* tokens del DS React, mapeados 1:1 a Bootstrap.
Sin lock-in
Solo CSS — sin frameworks JS requeridos. Funciona con cualquier backend.
Escalabilidad
Cambia el tema globalmente con solo editar tokens.css, sin tocar componentes.
